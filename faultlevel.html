Notes:
  • 3φ symmetrical fault: Isc = VLL / (√3 × Zcum), where Zcum accumulates source + phase-conductor impedances (one-way, positive-sequence).
  • 1φ-E fault: Isc = VLN / Zloop, where Zloop = source + phase conductor + CPC/earth conductor (one-way each). Assumes TN earthing system.
  • Peak factor κ = 1.02 + 0.98 × e−3R/X per IEC 60909. Computed from cumulative R/X at each node.
  • Device Status checks Isc 3φ sym against the specified kA breaking rating.
  • Results are indicative. Motor infeed, arc-flash, and IEC 60909 voltage-factor c adjustments require specialist software.
