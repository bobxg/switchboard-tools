1. Switchboard Sizer

Standards Referenced:
  • AS/NZS 3000:2018 - Electrical installations (Wiring Rules)
  • AS/NZS 3439.1 - Low-voltage switchgear and controlgear assemblies

Main Switch Rating Calculation

Main Switch Rating = Total Connected Load × Diversity Factor × Safety Factor Where: - Total Connected Load = Sum of all circuit breaker ratings - Diversity Factor = User-defined (typically 0.6-0.8 for residential, 0.8-1.0 for commercial) - Safety Factor = 1.25 (25% margin as per AS/NZS 3000)
Total Connected Load
Sum of all individual circuit ratings (in Amps)
Diversity Factor
Acknowledgment that not all circuits will be at full load simultaneously. Lower values indicate greater diversity.
Safety Factor
25% overhead to prevent nuisance tripping and allow for future expansion

Available Main Switch Sizes

Standard ratings rounded up to next available size: 63A, 80A, 100A, 125A, 160A, 200A, 250A, 315A, 400A, 500A, 630A, 800A

Busbar Sizing

Busbar Current Capacity = Main Switch Rating Width Selection: - ≤125A: 200mm width (standard) - >125A-250A: 300mm width - >250A-400A: 400mm width - >400A-630A: 600mm width - >630A: 800mm width
Note: Busbar sizing follows manufacturer specifications and AS/NZS 3439.1 for current density and temperature rise limits (typically 30-50K rise at rated current).

2. Cable Sizing Calculator

Standards Referenced:
  • AS/NZS 3008.1.1:2017 - Electrical installations - Selection of cables (Current-carrying capacity)
  • AS/NZS 3000:2018 - Electrical installations (Table C8 - Voltage drop)

Current Carrying Capacity

Required Cable Capacity = Design Current / (Installation Factor × Temperature Factor × Grouping Factor) Design Current = Load Current × 1.25 (AS/NZS 3000 requirement for continuous loads)
Installation Factor (AS/NZS 3008.1.1 Table 27/28)
Installation method derating:
  • Enclosed in conduit/trunking: Uses Table 27 Column 6 or 7
  • In air (on cable tray): Uses Table 27 Column 3 or 4
  • Direct buried: Uses Table 28
Temperature Factor (Table 27/28 notes)
Ambient temperature correction:
  • 25°C: 1.10
  • 30°C: 1.05
  • 35°C: 1.00 (base)
  • 40°C: 0.94
  • 45°C: 0.87
  • 50°C: 0.79
Grouping Factor (AS/NZS 3008.1.1 Table 22)
Multiple circuits in same enclosure:
  • 1 circuit: 1.00
  • 2 circuits: 0.80
  • 3 circuits: 0.70
  • 4-5 circuits: 0.65
  • 6-8 circuits: 0.60

Voltage Drop Calculation

Voltage Drop (V) = (mV/A/m) × Current × Length / 1000 Voltage Drop (%) = (Voltage Drop / Supply Voltage) × 100 AS/NZS 3000 Limits: - Final sub-circuits: 5% maximum - Mains cable: 1% maximum (from origin to main switchboard) - Total installation: 6% maximum
mV/A/m (millivolts per Amp per meter)
From AS/NZS 3000 Table C8. Values depend on:
  • Cable size (mm²)
  • Cable material (Copper or Aluminium)
  • System type (Single-phase or Three-phase)
  • Power factor (typically 0.8-1.0)
Current
Maximum expected load current (Amps)
Length
One-way cable run length (meters). Note: This is run length, not circuit length (no need to double)

Cable Selection Process

  1. Calculate derated current capacity requirement
  2. Select smallest cable from Table 27/28 that meets capacity
  3. Calculate voltage drop for selected cable
  4. If voltage drop exceeds limits, select next larger cable size
  5. Repeat until both capacity and voltage drop are compliant
Example: 32A load, 50m run, 3-phase, 40°C ambient, 3 circuits in conduit

Derated requirement = 32A / (1.0 × 0.94 × 0.70) = 48.6A
Selected: 10mm² copper (52A capacity from Table 27)
Voltage drop = 3.8 mV/A/m × 32A × 50m / 1000 = 6.08V
VD% = 6.08V / 400V × 100 = 1.52% ✓ Compliant

3. Voltage Drop Calculator (calc.html)

Standards Referenced:
  • AS/NZS 3000:2018 - Table C8 (Voltage drop for cables)
  • AS/NZS 3000:2018 - Table 4.1 (Maximum voltage drop limits)

Single-Phase Voltage Drop

VD (V) = 2 × I × L × R / 1000 Where: - VD = Voltage drop (volts) - I = Load current (amps) - L = One-way length (meters) - R = Conductor resistance per meter (mV/A/m from Table C8) - Factor of 2 accounts for both active and neutral conductors

Three-Phase Voltage Drop

VD (V) = √3 × I × L × R / 1000 Where: - √3 ≈ 1.732 (accounts for three-phase geometry) - Other variables same as single-phase
Resistance Values (R) - AS/NZS 3000 Table C8
Values in mV/A/m for copper cables at 75°C, PF=0.8:
  • 1.5mm²: 25 mV/A/m (1-phase), 22 mV/A/m (3-phase)
  • 2.5mm²: 15 mV/A/m (1-phase), 13 mV/A/m (3-phase)
  • 4mm²: 9.5 mV/A/m (1-phase), 8.2 mV/A/m (3-phase)
  • 6mm²: 6.4 mV/A/m (1-phase), 5.5 mV/A/m (3-phase)
  • 10mm²: 3.8 mV/A/m (1-phase), 3.3 mV/A/m (3-phase)
  • 16mm²: 2.4 mV/A/m (1-phase), 2.1 mV/A/m (3-phase)
  • 25mm²: 1.5 mV/A/m (1-phase), 1.3 mV/A/m (3-phase)
Compliance Limits (AS/NZS 3000 Table 4.1)
  • Mains: 1% maximum
  • Sub-mains: May utilize remaining voltage drop budget
  • Final sub-circuits: 5% maximum
  • Total (cumulative): 6% maximum from origin

4. Conduit Sizer

Standards Referenced:
  • AS/NZS 3000:2018 - Clause 3.9.4.2 (Cable fill for conduits and ducts)
  • AS/NZS 2053 - Conduits and fittings for electrical installations

Cable Fill Calculation

Fill Ratio = (Total Cable CSA) / (Conduit Internal CSA) AS/NZS 3000 Limits: - Single cable: 31% maximum fill - 2 cables: 31% maximum fill - 3+ cables: 40% maximum fill
Cable Cross-Sectional Area
Overall diameter including insulation and sheath:
  • 1.5mm² TPS: ~6.5mm diameter → 33.2mm² CSA
  • 2.5mm² TPS: ~7.5mm diameter → 44.2mm² CSA
  • 4mm² TPS: ~8.5mm diameter → 56.7mm² CSA
  • 6mm² TPS: ~10mm diameter → 78.5mm² CSA
  • Single core PVC: Calculate from (conductor + 2×insulation thickness)²×π/4
Conduit Internal Area
Based on nominal bore size (internal diameter):
  • 20mm: 314mm² internal area
  • 25mm: 491mm² internal area
  • 32mm: 804mm² internal area
  • 40mm: 1257mm² internal area
  • 50mm: 1963mm² internal area

Conduit Selection Process

  1. Count total number of cables to be installed
  2. Determine fill limit (31% for 1-2 cables, 40% for 3+ cables)
  3. Calculate total cable CSA (sum of all individual cable CSAs)
  4. Select smallest conduit where: Internal Area × Fill Limit ≥ Total Cable CSA
Example: 5 × 2.5mm² TPS cables (44.2mm² each)

Total cable CSA = 5 × 44.2mm² = 221mm²
Fill limit = 40% (more than 2 cables)
Required conduit area = 221mm² / 0.40 = 552.5mm²
Selected: 32mm conduit (804mm² internal area) ✓

5. Cable Weight Calculator

References:
  • Manufacturer specifications and datasheets
  • Material density calculations (Copper: 8.96 g/cm³, PVC: 1.4 g/cm³)

Weight Calculation Method

Cable Weight (kg/m) = (Conductor Weight + Insulation Weight + Sheath Weight) × Number of Cores Conductor Weight = CSA (mm²) × Length (m) × Copper Density (8.96 g/cm³) / 1000 Where CSA is the copper conductor cross-sectional area

Typical Cable Weights (kg/m)

Single Core PVC 0.6/1kV Copper
  • 1.5mm²: 0.028 kg/m
  • 2.5mm²: 0.041 kg/m
  • 4mm²: 0.059 kg/m
  • 6mm²: 0.082 kg/m
  • 10mm²: 0.133 kg/m
  • 16mm²: 0.206 kg/m
  • 25mm²: 0.318 kg/m
  • 35mm²: 0.437 kg/m
TPS (Twin & Earth) Copper
  • 1.5mm² TPS: 0.11 kg/m (2×1.5mm² + 1mm² earth)
  • 2.5mm² TPS: 0.17 kg/m (2×2.5mm² + 1.5mm² earth)
  • 4mm² TPS: 0.26 kg/m (2×4mm² + 2.5mm² earth)
  • 6mm² TPS: 0.37 kg/m (2×6mm² + 2.5mm² earth)

Cable Drum Load Calculation

Total Weight = Cable Weight per Meter × Drum Length + Drum Weight Typical drum weights: - Small drums (≤200m): 5-10 kg - Medium drums (200-500m): 10-20 kg - Large drums (500-1000m): 20-40 kg

6. Switchroom Sizer

Standards Referenced:
  • AS/NZS 3000:2018 - Clause 2.12 (Electrical equipment - Location and protection)
  • AS/NZS 3439.1 - Low-voltage switchgear assemblies (Clearance requirements)

Minimum Room Dimensions

Width = Equipment Width + 2 × Side Clearance (min 600mm each side) Depth = Equipment Depth + Front Access (min 1000mm) + Rear Clearance (min 600mm) Height = Equipment Height + Top Clearance (min 200mm) + Floor Mounting (min 150mm)
Front Working Space (AS/NZS 3000)
  • Normal access: 1000mm minimum
  • Restricted access: 700mm (only if egress available at both ends)
  • Service/maintenance access: 1200mm recommended
Side Clearances
  • For opening doors: 600mm minimum
  • For hinged panel access: 450mm minimum
  • Wall-mounted (no access): 50mm minimum
Rear Clearances
  • Rear access required: 800mm minimum
  • Cable entry only: 300mm minimum
  • No rear access: 150mm (for ventilation)
Ceiling Height
  • Standard switchboards: 2400mm minimum room height
  • Free-standing panels: Equipment height + 500mm minimum
  • Cable overhead runway: Add 600mm

Ventilation Requirements

Heat Dissipation (kW) = Switchboard Rating (kVA) × 0.02 to 0.03 Ventilation Area (m²) = Heat Dissipation (kW) / (Air Flow Rate × Temperature Rise) Typical values: - Natural ventilation: 0.5-1 m/s air flow - Forced ventilation: 2-4 m/s air flow - Temperature rise limit: 10-15K above ambient
Example: 630A main switch, 800mm wide switchboard

Width = 800mm + 2×600mm = 2000mm (2.0m)
Depth = 400mm (typical) + 1000mm + 600mm = 2000mm (2.0m)
Height = 2200mm + 200mm = 2400mm minimum ceiling
Room size: 2.0m × 2.0m × 2.4m high

7. Emergency Lighting Spacing

Standards Referenced:
  • AS/NZS 2293.1:2018 - Emergency escape lighting (System design, installation and operation)
  • AS/NZS 1680.0 - Interior and workplace lighting

Spacing Calculation (Lumen Method)

Maximum Spacing (m) = √[(Lumens × UF × MF) / (Required Lux × Width)] Where: - Lumens = Luminaire output (from manufacturer data) - UF = Utilization Factor (0.4 to 0.6, depends on reflectance) - MF = Maintenance Factor (0.8 for standard LED emergency lights) - Required Lux = From AS/NZS 2293.1 (0.5 to 10 lux) - Width = Corridor or area width (meters)
Utilization Factor (UF)
Accounts for light lost to walls/ceiling absorption:
  • Light surfaces (70% reflectance): UF = 0.5-0.6
  • Medium surfaces (50% reflectance): UF = 0.4-0.5
  • Dark surfaces (30% reflectance): UF = 0.3-0.4
Maintenance Factor (MF)
Accounts for aging and dirt accumulation:
  • LED luminaires, regular cleaning: MF = 0.8
  • LED luminaires, poor maintenance: MF = 0.7
Required Illuminance (AS/NZS 2293.1)
  • Escape routes: 1.0 lux (horizontal)
  • Open areas > 60m²: 0.5 lux
  • High risk task areas: 10 lux
  • Covered car parks: 1.0 lux

Mounting Height Constraint

Maximum Spacing ≤ 3 × Mounting Height This prevents excessive spacing that would create dark spots even if average illuminance is adequate. Based on photometric distribution patterns.
Example: 400 lumen LED, 2.5m high, 2m wide corridor, medium reflectance

UF = 0.45, MF = 0.8, Required = 1.0 lux
Spacing = √[(400 × 0.45 × 0.8) / (1.0 × 2)] = √[57.6] = 7.6m
Height constraint = 3 × 2.5m = 7.5m
Final spacing: 7.5m (limited by height) ✓

8. PV System Sizing

Standards Referenced:
  • AS/NZS 5033:2021 - Installation and safety requirements for photovoltaic arrays
  • AS/NZS 4777.1 - Grid connection of energy systems via inverters
  • Clean Energy Council Design Guidelines

Array Sizing Calculation

System Size (kW) = Daily Energy Use (kWh) / (Peak Sun Hours × System Efficiency) Where: - Daily Energy Use = Average daily consumption (kWh/day) - Peak Sun Hours = Location-specific (typically 4-5 hours in Australia/NZ) - System Efficiency = 0.75-0.80 (accounts for all losses)

Number of Panels

Number of Panels = System Size (W) / Panel Wattage (W) Round up to nearest whole number Example: 6500W system ÷ 400W panels = 16.25 → 17 panels

Inverter Sizing

Inverter Size = Array Size × (0.8 to 1.0) Ratio considerations: - 0.8-0.85: For hot climates or north-facing - 0.9-0.95: Standard residential - 1.0-1.1: For optimal orientation and cool climates
System Efficiency Factors
Overall efficiency (~75-80%) includes:
  • Inverter efficiency: 96-98%
  • DC cable losses: 2-3%
  • Temperature losses: 10-15%
  • Shading and soiling: 2-5%
  • Mismatch losses: 1-2%
Peak Sun Hours by Region
  • Northern Australia: 5.0-5.5 hours
  • Central Australia: 4.5-5.0 hours
  • Southern Australia: 4.0-4.5 hours
  • New Zealand North: 4.0-4.5 hours
  • New Zealand South: 3.5-4.0 hours

String Configuration (Voltage Calculations)

Panels per String = Inverter MPPT Voltage / Panel Vmp Verify: - Min Voltage: Panels × Vmp(min) ≥ Inverter MPPT Min - Max Voltage: Panels × Voc(max) ≤ Inverter Max Voltage Where Voc(max) = Voc × Temperature Coefficient × Lowest Temp
Example: 30 kWh/day usage, Sydney location

Peak sun hours = 4.5h
System efficiency = 0.78
System size = 30 / (4.5 × 0.78) = 8.55 kW
Using 400W panels: 8550 / 400 = 21.4 → 22 panels
Final system: 22 × 400W = 8.8kW array
Inverter: 8.8kW × 0.9 = 7.9kW → 8kW inverter ✓

11. Fault Level Calculator

Standards Referenced:
  • IEC 60909 — Short-circuit currents in three-phase AC systems
  • AS/NZS 3000:2018 — Clause 2.5.4–2.5.5
  • AS/NZS 3008.1.1 — Cable impedance data

Transformer Impedance (referred to LV)

Z_tx (Ω) = (%Z / 100) × V_LL² / S_VA R_tx = Z_tx / √(1 + (X/R)²) X_tx = (X/R) × R_tx

Upstream Network Impedance

Z_net = V_LL² / S_sc_VA (referred to LV side)

Symmetrical 3-Phase Fault Current

I_3ph (kA) = V_LL / (√3 × Z_cumulative) Peak factor (IEC 60909 Eq. 29): κ = 1.02 + 0.98 × e^(−3R/X) I_peak = κ × √2 × I_3ph

Single-Phase-to-Earth Fault

I_1ph = V_LN / Z_loop (phase impedance + CPC return path)
Cable resistance (mΩ/m at 75°C — AS/NZS 3008.1.1)
1.5mm²: 14.48 Cu / 23.69 Al  |  25mm²: 0.882 Cu / 1.454 Al  |  300mm²: 0.0762 Cu / 0.125 Al

13. Discrimination (Selectivity) Check

Standards Referenced:
  • AS/NZS 3000:2018 — Clause 2.5.5
  • IEC 61009 — Time-current characteristics

Overload Discrimination

Rating ratio = Upstream I_n / Downstream I_n ≥ 1.6 × — marginal discrimination likely ≥ 2.0 × — full discrimination achieved

Short-Circuit Discrimination

Instantaneous trip ranges: MCB Type B: 3–5 × I_n MCB Type C: 5–10 × I_n MCB Type D: 10–20 × I_n HRC Fuse: 2.0–4.5 × I_n (varies by rating) Condition: Upstream instantaneous trip threshold > Downstream instantaneous trip threshold
Note: Full discrimination requires upstream device to withstand fault long enough for downstream device to clear. Time-current curve comparison is the definitive check — this tool provides an indicative verdict only.

14. Transformer Sizing

Standards Referenced:
  • AS 2374.1-2 — Power transformers
  • AS/NZS 60076 — Losses and efficiency
  • AS/NZS 3000:2018 — Clause 4.14

Design Load

Design kVA = Total Load × Diversity Factor × (1 + Growth% / 100)

Temperature Derating

Dry-type: 1.0% per °C above 40°C ambient Oil (ONAN): 0.7% per °C above 40°C ambient

Secondary Full-Load Current

I_FLC (3-phase) = (kVA × 1000) / (V_LL × √3) I_FLC (1-phase) = (kVA × 1000) / V

Prospective Fault Current

I_fault = I_FLC / (Z_k / 100)

Estimated Losses (AS 2374.1)

No-load loss (W) ≈ kVA × (1.5 + kVA × 0.0005) Load loss (W) ≈ kVA × (5 + kVA × 0.002) Efficiency η = P_out / (P_out + Total Losses) × 100%
Typical %Z_k by rating
≤200 kVA: 4.0–4.5% (oil) / 4.5% (dry)  |  201–500 kVA: 4.5–5.0% / 5.0%  |  501–1000 kVA: 5.0–5.75% / 5.75%

17. Power Factor Correction (PFC)

Standards Referenced:
  • AS/NZS 2897 — Power factor correction equipment
  • AS/NZS 61921 — Power capacitors
  • IEC 62423 — Type B RCDs for capacitor banks

Required Reactive Power

φ₁ = arccos(PF_existing), φ₂ = arccos(PF_target) Q_required (kVAr) = P_kW × (tan φ₁ − tan φ₂) kVA reduction = kW/PF_existing − kW/PF_target

Resonance Risk Check

Resonant harmonic order: h_r = √(S_sc_kVA / Q_c_kVAr) If h_r coincides with a characteristic harmonic (5th, 7th, 11th…) → detuned reactor required Detuning: h_r ≤ 5: 7% reactor (tunes to 189 Hz) h_r ≤ 7: 5.67% reactor (tunes to 210 Hz)

Continuous Current Rating

Rating = R_reference × √(ΔT_allowed / 30°C) Multiple bars — proximity derating: 2 bars: × 0.85 | 3 bars: × 0.78 | 4 bars: × 0.72 Temperature rise check: ΔT_actual = ΔT_allowed × (I_actual / I_rated)²

Vertical Rod Resistance

R = (ρ / 2πL) × ln(4L / d) ρ = soil resistivity (Ω·m) L = rod length (m) d = rod diameter (m)

Horizontal Strip / Buried Tape

R = (ρ / πL) × ln(2L² / (w × d)) w = strip width (m), d = burial depth (m)

Earth Grid (Laurent-Niemann Approximation)

R_grid ≈ ρ / (4√A) + ρ / L_total A = grid area (m²), L_total = total conductor length (m)

Parallel Electrodes

R_combined = R_single / (n × η) Efficiency factor η accounts for mutual coupling between electrodes (Schwarz method)

24. Lux Calculator (Lumen Method)

Standards Referenced:
  • AS/NZS 1680.1 — Interior and workplace lighting
  • NCC Section J6 — Artificial lighting energy efficiency

Number of Luminaires

N = (E × A) / (Φ × UF × MF) E = target illuminance (lux) A = room area (m²) Φ = luminous flux per luminaire (lm) UF = utilisation factor MF = maintenance factor (typically 0.8 for LED)

Room Index

k = (L × W) / (H_m × (L + W)) H_m = mounting height above work plane (m)

Utilisation Factor (approximate)

UF ≈ 0.30 + 0.40 × (1 − e^(−0.8k)) + reflectance correction

Spacing-to-Height Ratio

Maximum spacing = SHR × H_m Typical SHR for LED troffers: 1.2–1.5
