# Switchboard & Electrical Tools

A collection of web-based calculators and tools for electrical design and verification, compliant with Australian Standards (AS/NZS).

## Tools Available

### Main Tools
- **Switchboard Sizer** - Interactive MSB layout designer with drag-drop modules
- **Switchroom Sizer** - Calculate minimum switchroom dimensions per AS/NZS 3000
- **Lighting Search** - Search approved lighting manufacturers and suppliers
- **Verification** - Documentation of calculation methods and standards compliance

### TBC Tools
- **Conduit Sizer** - AS/NZS 3000:2018 compliant conduit sizing with correct fill factors
- **Cable Sizing** - Cable selection based on AS/NZS 3008.1.1 with derating
- **Cable Weight** - Calculate cable weights with manufacturer data references
- **PV Sizing** - Solar PV system cable and protection calculations
- **Emergency Lighting** - AS/NZS 2293 compliant emergency lighting calculations
- **Site Inspection** - Inspection checklist and documentation tool

## Standards Compliance

All calculators follow current Australian and New Zealand standards:
- AS/NZS 3000:2018 - Electrical installations (Wiring Rules)
- AS/NZS 3008.1.1 - Cable sizing and selection
- AS/NZS 2293 - Emergency lighting
- AS/NZS 3439.1 - Switchgear and controlgear assemblies
- AS/NZS 5000.1 - Electric cables - Polymeric insulated
- AS 2053 - Conduits and fittings

## Usage

Open `index.html` in a web browser to access all tools.
